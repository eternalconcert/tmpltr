Context variables

↑
The context is a JavaScript object that contains variables which can be used in the template. If you pass this object to the template 
    
    {
        firstName: 'Jack',
        lastName: 'Shephard',
        status: 'lost',
    }
you can use it like this in the template: 
    
    <h3>Passenger information:</h3>
    <div>Name: {{ lastName }}, {{ firstName }}</div>
    <div>Status: {{ status }}</div>
The final result will be this:

Passenger information:

Name: {{ lastName }}, {{ firstName }}
Status: {{ status }}

Context loops

↑
Beside the possibility to inject simple variables, you can also pass arrays (of values of simple types) to the template. In the template you can iterate those array: 
    
    { passengers:
      [
        'Jack',
        'John',
        'Kate',
        'Sawyer',
        'Hurley',
        'Sayid',
        'Charlie',
        'Claire',
        'Michael',
        'Shannon',
        'Jin-Soo',
        'Sun-Hwa',
        'Boone',
        'Walter',
      ]
    };
you can use it like this in the template: 
  
    <h3>Passengers</h3>
    <ol>
      {% for passenger in passengers %}
        <li>{{ passenger }}</li>
      {% endfor %}
    </ol>
This renders to:

Passengers

    {% for passenger in passengers %}
  1. {{ passenger }}
    2. {% endfor %}

From DB

{% for item in dbItems %} {{ item }}
{% endfor %}
