TMPLTR offers the ability to server side render HTML pages and use the full power of JavaScript. It is heavy inspired by Flask and Jinja.

If your are familiar with these awesome pieces of software, learning TMPLTR will be no big deal for you.

Examples

Context variables

The context is an JavaScript object that contains variables which can be used in the template. If you pass this object to the template 
    { firstName: 'Jack', lastName: 'Shephard', status: 'lost' }
you can use it like this in the template: 
    <h3>Passenger information:</h3>
    <div>Name: {{ lastName }}, {{ firstName }}</div>
    <div>Status: {{ status }}</div>
The final result will be this:

Passenger information:

Name: {{ lastName }}, {{ firstName }}
Status: {{ status }}
Counter: {{ counter }}
